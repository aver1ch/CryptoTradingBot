# CryptoTradingBot

Простой учебный крипто‑бот на Python: получает цену BTC/USDT с TradingView через Selenium и торгует по пересечению двух SMA (300 и 900). В проект также встроен Telegram‑бот для просмотра статуса.

## Что внутри
- `main.py` — запуск торгового цикла.
- `trading_processes.py` — логика стратегии (SMA 300/900), комиссии и запись состояния.
- `get_price.py` — парсинг текущей цены с TradingView.
- `indicators.py` — расчёт SMA.
- `io_json.py` — чтение/запись состояния в JSON.
- `bot.py` — Telegram‑бот для мониторинга.
- `params.json` — текущее состояние стратегии (баланс, цена покупки, флаг сделки).
- `system.json` — системные параметры (старт/текущее время, флаг панической продажи).

## Предупреждение
- Это экспериментальный код. Используйте только на тестовых данных/средствах.
- В `bot.py` есть место для API‑токена Telegram. Никогда не храните токены в открытом виде в публичных репозиториях.

## Требования
- Python 3.9+
- Google Chrome
- Библиотеки: `selenium`, `webdriver-manager`, `pyTelegramBotAPI`

Установка зависимостей:
```bash
pip install selenium webdriver-manager pyTelegramBotAPI
```

## Быстрый старт
1. Проверьте, что Chrome установлен.
2. При необходимости отредактируйте стартовые значения в `params.json`.
3. Запустите торговый цикл:
```bash
python main.py
```

## Telegram‑бот
1. В `bot.py` задайте токен:
   - Найдите строку `bot = telebot.TeleBot()` и передайте токен.
2. (Опционально) Добавьте свой `peerID` в список в обработчике сообщений, чтобы ограничить доступ.
3. Запустите:
```bash
python bot.py
```

## Как работает стратегия
- Цена берётся с TradingView (`BINANCE:BTCUSDT`) через Selenium.
- Строятся две SMA: 300 и 900 последних значений.
- Условие покупки: SMA300 > SMA900 и нет открытой сделки.
- Условие продажи: сделка открыта и текущая потенциальная прибыль превышает комиссию.
- Состояние сделки и баланса пишется в `params.json`.

## Примечания
- Скрейпинг TradingView может ломаться при изменении разметки.
- Сейчас лог транзакций в `bot.py` читает `transactions.txt`, но файл не создаётся автоматически.
